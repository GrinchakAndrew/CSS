Rendering-12:

How Browsers Apply Styles

  		•

    CSS Browser Rendering:
    • (1) External CSS is applied first, followed by the HTML-embedded styles
    • (2) Finally, HTML inline styles are applied last:
    • (3) If any of the 3 types of styles conflict with each other, the last one will kick in to override the rest.
    • (4) The last rules applied WINS!

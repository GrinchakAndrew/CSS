    CSS Specificity Rules Outlined:
    • (1) ID selectors have a higher specificity than attribute selectors. For example, selector '#p123' is more specific than '[id=p123]' in terms of the cascade:
    • The first rule (A) is more specific than the second one (B)
    • (2) Contextual selectors are more specific than a single element selector. Selectors involving more than one HTML element selector have greater weight.
    • (3) The embedded style sheet is closer to the element to be styled, and has greater weight.
    • (4) The last rule defined overrides any previous, conflicting rules.
    • (5) A class selector beats any number of element selectors. '.introduction' is more important than 'html body div div h2 p'
    • (6) The universal selector has a specificity of 0, 0, 0, 0 . '*', 'body *' and similar selectors have a zero specificity. Inherited values also have a specificity of 0, 0, 0, 0.

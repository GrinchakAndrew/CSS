    CSS Selectors, Details:
    • (1) Global Selectors affect every element, as addressed by the Selector
    • (2) Class Selectors affect every element, as addressed by its Class
    • (3) Class Selector is an HTML attribute, usable for as many elements as needed:
    • (4) Class Selector identifies a group of elements on a Page by a '.' sign:
    • (5) ID Selector identifies a individual element on a Page by a '#' sign:
    • (6) ID Selector MUST be unique to the Page!
    • (7) Pseudo-classes are special classes that are used to define the behavior of HTML elements. They are used to add special effects to some selectors, which are applied automatically in certain states. For example: ':visited'
    • (8) Pseudo-elements help assign style to content that does not exist in the source document. For example: ':first-line' or ':after'

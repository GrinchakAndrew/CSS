Text Formatting-19

  		•

    CSS Text Formatting:
    • 'text-align' - text alignment inside an element: (left, right, center, justify)
    • 'text-decoration' - text decoration (underline, overline, line-through, blink)
    • 'letter-spacing' - interval between symbols (normal, lenght: 5px);
    • 'white-space' - wrap or not to wrap text inside an element (normal|nowrap|pre|pre-line|pre-wrap|initial|inherit) - 'nowrap' - text will not break untill '< br >' is found - 'normal' - white space is allowed between words - 'pre' - white spaces are not limited in number (usually used to format code layouts)

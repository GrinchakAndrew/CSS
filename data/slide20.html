    CSS Font Properties:
    • 'Font-size' -
    • 'Fixed Units: Pixels' - show the set size independent of the Device \ User Agent.
    • 'Relative Units: em' - show the set size relative to the device environment.
    • '1em' - tells the device to show the font-zise to the 100% of the Default Device font-size. Below is an example of styling headers font-size for different-size tags, where h1,h2,h3,h4,h5 tags are relative of each other:
    • 'Font-weight, font-style:' -
    • Value Description Play it
      normal The browser displays a normal font style. This is default Play it »
      italic The browser displays an italic font style Play it »
      oblique The browser displays an oblique font style Play it »

    • 'Font-variant:' - small capitalization against default styling
      Value Description Play it
      normal The browser displays a normal font. This is default Play it »
      small-caps The browser displays a small-caps font Play it »
      initial Sets this property to its default value. Read about initial Play it »

    • 'text-transform:' - property controls the capitalization of text.
      Value Description Play it
      none No capitalization. The text renders as it is. This is default Play it »
      capitalize Transforms the first character of each word to uppercase Play it »
      uppercase Transforms all characters to uppercase Play it »
      lowercase Transforms all characters to lowercase Play it »
      initial Sets this property to its default value. Read about initial Play it »
      inherit Inherits this property from its parent element. Read about inherit
    • 'text-indent:' - property controls the indentation of text. Can be set to a negative value in order to hide text.
      Value Description Play it
      length Defines a fixed indentation in px, pt, cm, em, etc. Default value is 0 Play it »
      % Defines the indentation in % of the width of the parent element Play it »
      initial Sets this property to its default value. Read about initial Play it »
      inherit Inherits this property from its parent element. Read about inherit
    • 'line-height:' - property controls the indentation of text. Can be set to a negative value in order to hide text. It is the only CSS property that allows use of no units, i.e., a 'multiple': just a number to relatively set the line-height against the font-size already used on a webpage.
      Value Description Play it
      normal A normal line height. This is default Play it »
      number A number that will be multiplied with the current font size to set the line height Play it »
      length A fixed line height in px, pt, cm, etc. Play it »
      % A line height in percent of the current font size Play it »
